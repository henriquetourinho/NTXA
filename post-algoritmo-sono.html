O Algoritmo do Sono:
Decodificando a cronobiologia com Python.

Henrique Tourinho

CTO & Founder @ NXTA

A maioria dos "trackers" de sono (smartwatches) comete um erro fundamental: eles são reativos. Eles te dizem na manhã seguinte que você dormiu mal. Na NXTA, acreditamos que isso é inútil.

O BioClock foi construído sobre uma premissa diferente: Previsibilidade. Usando dados biométricos em tempo real, queremos te dizer exatamente quando ir para a cama para maximizar a liberação de Hormônio do Crescimento (GH) e a limpeza glicofática do cérebro.

O Modelo de Dois Processos

Para construir o algoritmo do BioClock, baseamo-nos no modelo científico de Borbély (1982), que define o sono como a interação de duas variáveis:

  • Processo S (Pressão do Sono): A acumulação homeostática de adenosina no cérebro. Aumenta quanto mais tempo você fica acordado.
  • Processo C (Ritmo Circadiano): Uma onda senoidal controlada pelo núcleo supraquiasmático, que dita quando você deve estar alerta ou sonolento, independente do cansaço.

Traduzindo Biologia para Código

O maior inimigo do Processo C é a luz artificial (espectro azul) à noite. Ela suprime a melatonina, "enganando" seu cérebro de que ainda é dia.

Abaixo está um trecho da classe CircadianEngine do BioClock. Ela calcula o DLMO (Dim Light Melatonin Onset) ajustado pela exposição a luz (Lux) capturada pelo sensor do dispositivo.

core/circadian_engine.py 
from datetime import datetime, timedelta

class CircadianEngine:
    def __init__(self, chronotype: str, wake_time: datetime):
        self.chronotype = chronotype  # 'lion', 'bear', 'wolf'
        self.wake_time = wake_time

    def calculate_melatonin_onset(self, current_lux: float) -> datetime:
        # Base DLMO (Dim Light Melatonin Onset)
        base_onset = self.wake_time + timedelta(hours=14)
        
        # Penalidade por Luz Azul (Blue Light Penalty)
        # Se exposto a > 100 lux (tela de celular) após o por do sol
        delay_minutes = 0
        
        if current_lux > 100.0:
            # Modelo linear: 10 min de atraso para cada 15 min de exposição
            delay_minutes = 90 

        adjusted_onset = base_onset + timedelta(minutes=delay_minutes)
        
        return adjusted_onset

Intervenção, não Observação

Com base nesse cálculo, o BioClock não espera você dormir. Às 19:30, ele envia uma notificação "Push" agressiva:

"Detectamos alta luminosidade. Seu pico de melatonina será atrasado em 90 minutos. Bloqueie a luz azul agora para recuperar seu ciclo."

Isso é engenharia aplicada à saúde. Transformamos dados passivos em ações corretivas imediatas.
